Browser Cache Debug

1. Öffne DevTools (F12)

2. Gehe zum "Network" Tab

3. Aktiviere "Disable cache"

4. Hard Reload: Ctrl+Shift+R (Windows) oder Cmd+Shift+R (Mac)

5. Check ob index-Dp5HWYXR.js geladen wird (neue Version)

Alternative: Cache komplett leeren

1. DevTools → Application Tab → Storage → "Clear site data"

2. Oder: Browser Settings → Clear browsing data → Cached images and files
